# Meowroom

This project is a website for the "Meowroom" cat shelter and "BlueCross for Cats" foundation.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

1.  Clone the repository.
2.  Run `npm install`.
3.  Run `npm run dev` to start the development server.

## Features

- Home page with hero section and mission statement.
- Cats gallery with filtering.
- "About Us" page with the shelter's story and video.
- Donation page.
- Visit booking information.
- Contact information.
